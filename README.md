# AI-Operator
AI Operator เป็นปัญญาประดิษฐ์เพื่อบริการประชาชน พัฒนาโดย ศูนย์บริการร่วม กระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม (ศบร.ทส.) ทำหน้าที่ตอบคำถามประชาชนเกี่ยวกับงานอนุมัติ อนุญาต และงานบริการอื่น ๆ ของกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม (ทส.) โดยในระยะแรกนี้ เป็น AI ต้นแบบ สามารถตอบคำถามที่ถามบ่อย (FAQ) จำนวน 30 คำถามก่อน และจะสอน AI (supervised-training) ให้จำคำถามที่น่าสนใจและคำตอบที่กระชับ ชัดเจน และถูกต้อง โดย AI นี้ สามารถจำคำตาม-คำถาม ได้จำนวนมากอย่างไร้ขีดจำกัด และสามารถนำสู่คำตอบได้อย่างถูกต้อง รวดเร็ว และเป็นมาตรฐานเดียวกัน
